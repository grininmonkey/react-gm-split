# react-gm-split

## Install

```bash
npm install --save react-gm-split
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-gm-split'
import 'react-gm-split/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
